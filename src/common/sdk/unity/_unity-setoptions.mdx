After instantiation, within your `Start()` function, set up the Web3Auth Options as follows:

```cs
web3Auth.setOptions(new Web3AuthOptions()
{
  clientId = "BEglQSgt4cUWcj6SKRdu5QkOXTsePmMcusG5EAoyjyOYKlVRjIF1iCNnMOTfpzCiunHRrMui8TIwQPXdkQ8Yxuk",
  redirectUrl = new System.Uri("torusapp://com.torus.Web3AuthUnity/auth"),
  network = Web3Auth.Network.CYAN,
});
```

:::tip

You can also configure your `client_id`, `redirect_url` and `network` within the script setting in Unity Editor. This overrides the values you pass in
the `setOptions` function.
