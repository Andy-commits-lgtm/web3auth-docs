#### `configureAdapter(ADAPTER)`

import TabItem from "@theme/TabItem";
import Tabs from "@theme/Tabs";

To configure an adapter, create the instance of the adapter by using its corresponding package and
pass the returned `adapter` instance in the `configureAdapter` function.

:::tip

Refer to the [Adapters documentation](/sdk/pnp/web/adapters) to know more deeply about what adapters
are available and how to configure them.

:::

<TabItem value="evm">

If you want to support All EVM External Wallets, your should use the Default External Adapter for
EVM. This adapter supports MIPD (EIP6163) so all available wallets in the browser will automatically
be detected as well. Apart from EIP6163, it also provides an option to use the WalletConnect v2
wallet list.

- Import the `getDefaultExternalAdapters` from `@web3auth/default-evm-adapter` package
- Get all the adapters in the `adapters` variable as an array of `IAdapter`.
- Iterate over the `adapters` array and configure each adapter using `configureAdapter` function.

```javascript
import { getDefaultExternalAdapters } from "@web3auth/default-evm-adapter";

const adapters = await getDefaultExternalAdapters({
  options: {
    clientId,
    web3AuthNetwork: WEB3AUTH_NETWORK.SAPPHIRE_MAINNET,
    privateKeyProvider,
  },
});

adapters.forEach((adapter) => {
  web3auth.configureAdapter(adapter);
});
```
