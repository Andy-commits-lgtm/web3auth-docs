---
title: Install Web3Auth
---

Install `@web3auth/core`, `@web3auth/base` & `@web3auth/openlogin-adapter` using either [npm](https://www.npmjs.com/package/@web3auth/core) or
[yarn](https://yarnpkg.com/package/@web3auth/core):

```bash npm2yarn
npm install --save @web3auth/core @web3auth/base @web3auth/openlogin-adapter
```
