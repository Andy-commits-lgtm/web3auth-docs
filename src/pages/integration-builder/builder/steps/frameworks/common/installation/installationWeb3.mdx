title: Install web3.js
---

import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

Install `web3.js` using either [npm](https://www.npmjs.com/package/web3) or [yarn](https://yarnpkg.com/package/web3):

<Tabs
  defaultValue="npm"
  values={[
    { label: "npm", value: "npm" },
    { label: "yarn", value: "yarn" },
  ]}
>
<TabItem value="npm">

```shell
npm install --save web3
```

</TabItem>

<TabItem value="yarn">

```shell
yarn add web3
```

</TabItem>

</Tabs>

:::tip

Add the file `web3RPC.ts` to the project. This file contains the code to interact with the blockchain.

:::
