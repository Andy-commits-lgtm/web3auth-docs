---
title: Install Web3Auth
---

Install `@web3auth/core` using a CDN:

```html
<script src="https://cdn.jsdelivr.net/npm/@web3auth/core@latest/dist/core.umd.min.js"></script>
```

[![Npm package monthly downloads](https://badgen.net/npm/dm/@web3auth/core)](https://www.npmjs.com/package/@web3auth/core)
[![](https://data.jsdelivr.com/v1/package/npm/@web3auth/core/badge)](https://www.jsdelivr.com/package/npm/@web3auth/core)

This is the main Plug and Play No Modal package that contains the Web3AuthCore SDK helping you login users with their social accounts and blockchain
wallets, according to the chosen adapter. You have the flexibility of making your own UI and Web3Auth sits in the background.

:::warning Update to the Latest-SDK

Please use the **[latest version](https://www.npmjs.com/package/@web3auth/core)** of Web3Auth SDKs.

:::
