## Supported Browsers List

Web3Auth SDKs have functions which require BigInt compatability and thus the browsers which are supported are as follows. The browsers are listed
alongside their versions which are supported.

- Chrome 67+
- Chrome for Android 107+
- Safari 14+
- Safari on iOS 14+
- Edge 79+
- Firefox 68+
- Firefox for Android 106+
- Opera 54+
- Opera Mobile 72+
- Samsung Internet 9.2+
- UC Browser for Android 13.4+
- QQ Browser 13.1+
- Android Browser 107+

:::note source code

When creating your React App in production, make sure to follow the steps described [here](./react-big-int-error) to avoid BigInt Error.

:::
