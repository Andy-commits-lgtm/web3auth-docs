This document is your quick start guide to integrating Web3Auth into your wallet/dapp. Integrating Web3Auth is easy and you have multiple ways to do
it according to your preference:

- You can start by [Integrating a basic Web3Auth Modal](#integrate-web3auth-in-4-simple-steps) into your application and further customize it to your
  needs.

- You also have the choice to check out our [Integration Builder](/integration-builder), which helps you develop a quick start boilerplate
  application within seconds.

- Further, you can checkout our [Guides](/guides) and tutorials around [Connecting your Blockchain](/connect-blockchain/) and
  [Custom Authentication](/custom-authentication/) to make Web3Auth suit your needs.

- Side note: if you're a pro developer, just hop on to our [SDK Reference](/sdk) and get started!

## Integrate Web3Auth in 4 Simple Steps

---

Integrating Web3Auth into your project is as simple as following the basic 4 steps written below.

### 1. Install Web3Auth

Import the Web3Auth package in your project. If you're using any other platform, check our respective [SDK Reference](/sdk).

```bash npm2yarn
npm install --save @web3auth/modal
```

### 2. Get your Client ID from Web3Auth Dashboard

Hop on to the [Web3Auth Dashboard](https://dashboard.web3auth.io) and create a new project. Use the Client ID of the project to start your
integration.

<img src={Image5} style={{ alignSelf: "center", maxWidth: "100%" }} />

### 3. Initialize Web3Auth for your preferred blockchain

Web3Auth needs to initialise as soon as your app loads up to enable the user to log in. Preferably done within a constructor, initialisation is the
step where you can pass on all the configurations for Web3Auth you want. A simple integration for Ethereum and Solana blockchains will look like this:

<Tabs
  defaultValue="ethereum"
  values={[
    { label: "Ethereum", value: "ethereum" },
    { label: "Solana", value: "solana" },
  ]}
>

<TabItem value="ethereum">

```js
import { Web3Auth } from "@web3auth/modal";

//Initialize within your constructor
const web3auth = new Web3Auth({
  clientId: "", // Get your Client ID from Web3Auth Dashboard
  chainConfig: {
    chainNamespace: "eip155",
    chainId: "0x1",
  },
});

await web3auth.initModal();
```

</TabItem>

<TabItem value="solana">

```js
import { Web3Auth } from "@web3auth/modal";

//Initialize within your constructor
const web3auth = new Web3Auth({
  clientId: "", // Get your Client ID from Web3Auth Dashboard
  chainConfig: {
    chainNamespace: "solana",
    chainId: "0x1", // Please use 0x1 for Mainnet, 0x2 for Testnet, 0x3 for Devnet
  },
});

await web3auth.initModal();
```

</TabItem>

</Tabs>

### 4. Login your User

Once you're done initialising, just create a button that triggers to open the login modal for the user on their request. Logging in is as easy as:

```js
await web3auth.connect();
```

:::caution

If you're new to Web3 development, you might be facing certain issues with Webpack while configuring your application. Please refer to our
[Webpack Issues Troubleshooting Guide](/troubleshooting/webpack-issues) for more information.

:::
