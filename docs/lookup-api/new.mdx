# Wallet Pregeneration API

## Overview

The Wallet Pregeneration API is a simple API that allows you to pre-generate wallets for your users. This is useful if you want to create a wallet for
your users before they sign up for your service or retrieve the address of pre-existing wallets.

## API

### HTTP Request

`GET https://lookup.web3auth.io/lookup`

### Query Parameters

| Parameter         | Description                  | Example                                                                                   |
| ----------------- | ---------------------------- | ----------------------------------------------------------------------------------------- |
| `verifier`        | The verifier name            | `w3a-google-demo`                                                                         |
| `verifierId`      | The verifier ID value        | `yash@tor.us`                                                                             |
| `web3AuthNetwork` | Name of the Web3Auth network | `sapphire_mainnet`                                                                        |
| `clientId`        | The client ID                | `BPi5PB_UiIZ-cPz1GtV5i1I2iOSOHuimiXBI0e-Oe_u6X3oVAbCiAZOTEBtTXw4tsluTITPqA8zMsfxIKMjiqNQ` |

### Sample Request

```bash
GET https://lookup.web3auth.io/lookup?verifier=w3a-google-demo&verifierId=yash@tor.us&web3AuthNetwork=sapphire_mainnet&clientId=BPi5PB_UiIZ-cPz1GtV5i1I2iOSOHuimiXBI0e-Oe_u6X3oVAbCiAZOTEBtTXw4tsluTITPqA8zMsfxIKMjiqNQ
```
