## Installation

### Swift Package Manager

To add a package dependency when using the Swift Package Manager, simply navigate to the following option in Xcode:

`File > Swift Packages > Add Package Dependency`

In the Choose Package Repository prompt add this url:

```sh
https://github.com/torusresearch/tkey-rust-ios/releases/tag/0.0.4
```

### Cocoapods

```sh
pod 'tkey-rust-ios', '~> 0.0.4'
```
