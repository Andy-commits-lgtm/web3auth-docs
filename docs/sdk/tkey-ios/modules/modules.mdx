Modules extend the tKey iOS SDK's capability to provide additional functionality for managing your tKey shares. These modules can be used for multiple
purposes, from generating seed phrases to generating password or converting them into a form that can remembered or sent out.

### Private Keys Module

[`PrivateKeysModule`](/sdk/tkey-ios/modules/private-keys): Adds the capability to store extra private keys on the metadata.

### Security Question Module

[`SecurityQuestionModule`](/sdk/tkey-ios/modules/security-questions): Adds the capability to add or remove a question and password as a share.

### Seed Phrase Module

[`SeedPhraseModule`](/sdk/tkey-ios/modules/seed-phrase): Adds the capability to store and use seedphrases on the metadata.

### Share Serialization Module

[`ShareSerializationModule`](/sdk/tkey-ios/modules/share-serialization): Adds the capability to import/ export a share from `tKey`.

### Share Transfer Module

[`ShareTransferModule`](/sdk/tkey-ios/modules/share-transfer): Adds the capability to transfer a share to another device.
