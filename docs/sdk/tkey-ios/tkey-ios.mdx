The Web3Auth [tKey iOS SDK](https://github.com/torusresearch/tkey-rust-ios) is a client-side library you can use with your iOS app to authenticate
users using Web3Auth. This authentication can be achieved using any social logins Web3Auth provides or a custom authentication flow. For using
Web3Auth's tKey natively in iOS, Web3Auth provides a tKey iOS SDK written in Swift.

### Requirements

- iOS 13+
- Xcode 11.4+ / 12.x
- Swift 4.x / 5.x

:::note source code

Source code of the SDK can be found **[on GitHub](https://github.com/torusresearch/tkey-rust-ios)**.

:::

:::info Quick Start

You can have a look at the **[example](https://github.com/torusresearch/tkey-rust-ios-example)** made on top of this SDK and try it out yourself.

:::

## Installation

### Swift Package Manager

To add a package dependency when using the Swift Package Manager, simply navigate to the following option in Xcode:

`File > Swift Packages > Add Package Dependency`

In the Choose Package Repository prompt add this url:

```
https://github.com/torusresearch/tkey-rust-ios
```

### Cocoapods

```sh
pod install tkey-rust-ios
```
