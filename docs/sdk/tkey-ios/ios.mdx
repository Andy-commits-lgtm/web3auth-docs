For using Web3Auth's tKey natively in iOS, Web3Auth provides a [tKey iOS SDK](https://github.com/torusresearch/tkey-rust-ios), written in Swift. The
Web3Auth tKey iOS SDK is a client-side library you can use with your iOS app to authenticate users using Web3Auth. This authentication can be achieved
by using any of the social logins Web3Auth provides or using a custom authentication flow of your choice.

### Requirements

- iOS 13+
- Xcode 11.4+ / 12.x
- Swift 4.x / 5.x

:::note source code

Source code of the SDK can be found **[on GitHub](https://github.com/torusresearch/tkey-rust-ios)**.

:::

:::info Quick Start

You can have a look at the **[example](https://github.com/torusresearch/tkey-rust-ios-example)** made on top of this SDK and try it out yourself.

:::

## Installation

### Swift Package Manager

If you are using the Swift Package Manager, open the following menu item in Xcode:

`File > Swift Packages > Add Package Dependency`

In the Choose Package Repository prompt add this url:

```
https://github.com/torusresearch/tkey-rust-ios
```

### Cocoapods

```sh
pod install tkey-rust-ios
```
