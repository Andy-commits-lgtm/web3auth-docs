## Overview

While using the Web3Auth Web SDK, you get an EIP1193-compatible provider, similar to the MetaMask
provider. This provider can be used with libraries to make blockchain calls such as getting the
user's account, fetching balances, signing transactions, sending transactions, etc. We have
highlighted a few examples to get you started quickly.

---

## Installation

To interact with the XDC Network, you can use the Web3Auth provider and XDC.

### NPM Installation

```bash
npm install @web3auth/no-modal @web3auth/ethereum-provider @web3auth/auth-adapter @web3auth/wallet-services-plugin
```

## Initializing and Instantiating the Web3Auth SDK

We will now initialize Web3Auth with the XDC configuration.

```typescript
import { Web3AuthNoModal } from "@web3auth/no-modal";
import { EthereumPrivateKeyProvider } from "@web3auth/ethereum-provider";
import { AuthAdapter } from "@web3auth/auth-adapter";
import { XDC_MAINNET, XDC_APOTHEM_NETWORK, XDC_DEVNET } from "./chainConfig";
import XdcRpc from "./xdcRpc";

const clientId = "YOUR_CLIENT_ID"; // Replace with your Web3Auth client ID

const privateKeyProvider = new EthereumPrivateKeyProvider({
  config: { chainConfig: XDC_MAINNET },
});

const web3auth = new Web3AuthNoModal({
  clientId, // Your Web3Auth Client ID
  chainConfig: XDC_MAINNET, // Using XDC Mainnet
  privateKeyProvider,
});

const authAdapter = new AuthAdapter({
  adapterSettings: {
    uxMode: "redirect",
  },
});
web3auth.configureAdapter(authAdapter);
await web3auth.init();
```

## Getting the Web3Auth Provider

After initializing Web3Auth, the next step is to authenticate and retrieve the provider to interact
with the XDC blockchain.

```typescript
await web3auth.connect();

// Get the provider
const provider = web3auth.provider;
```

Once logged in, Web3Auth provides a JWT token and access to the XDC provider.

## Fetching User Info

After logging in, you can retrieve information about the user.

```typescript
const user = await web3auth.getUserInfo();
console.log(user);
```

## Get Account and Balance

Once logged in, we can use `xdcRpc.ts` to fetch the userâ€™s account and balance.

```typescript
// xdcRpc.ts should be implemented as per previous example
const xdcRpc = new XdcRpc(provider);
await xdcRpc.init();

// Get the user's XDC account address
const accounts = await xdcRpc.getAccounts();
console.log("Account: ", accounts[0]);

// Get the balance
const balance = await xdcRpc.getBalance();
console.log("Balance: ", balance);
```

## Send Transaction

You can send a transaction using the user's private key.

```typescript
const transaction = await xdcRpc.sendTransaction();
console.log("Transaction Hash: ", transaction);
```

This will send a transaction from the user's XDC account.

## Sign a Message

You can also sign a message using the private key of the logged-in user.

```typescript
const signedMessage = await xdcRpc.signMessage("Hello XDC!");
console.log("Signed Message: ", signedMessage);
```

This will return the signature of the message.

## Conclusion

With Web3Auth, you can easily integrate XDC blockchain capabilities into your application. By using
the XDCWeb library, you can perform various blockchain operations such as fetching accounts, sending
transactions, signing messages, and interacting with smart contracts.
