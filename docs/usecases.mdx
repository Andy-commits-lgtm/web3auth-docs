Welcome! If you are new to Web3Auth, you are in the right place. Here we will cover how to get started using Web3Auth's Plug, Play n Customize SDK.

There are only a few steps you have to complete to start using Auth0.

1. Sign up and create Web3Auth account [here](https://developer.tor.us/)

2. Based on your app's stack and usecase, choose one of our guides (for [wallets](/guides#wallets) or [dapps](/guides#dapps)), and hook it up to your app.

3. Include final touches by [connecting to a particular blockchain](/guides#blockchains), and/or whitelabeling the UI
