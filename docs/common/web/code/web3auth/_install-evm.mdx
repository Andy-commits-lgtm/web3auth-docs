<InstallWeb3Auth/>

- #### If you want to use default configured adapters , install evm-adapter-factory

    ```shell
        npm i --save @web3auth/evm-adapter-factory
    ```

- #### If you want to configure adapters , install individual adapter packages which you want to use:-
- For example if you want to use openlogin, metamask, torus wallet and wallet connect v1 then install following packages:-
    ```shell
        npm i --save @web3auth/openlogin-adapter
        npm i --save @web3auth/metamask-adapter
        npm i --save @web3auth/torus-evm-adapter
        npm i --save @web3auth/wallet-connect-v1-adapter
    ```
