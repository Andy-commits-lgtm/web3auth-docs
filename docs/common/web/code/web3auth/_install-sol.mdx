#### If you want to use default configured adapters , install solana-adapter-factory

```shell
    npm i --save @web3auth/solana-adapter-factory
```

#### If you want to configure adapters , install individual adapter packages which you want to use:-

    - For example if you want to use openlogin, phantom and torus solana wallet, install following packages:-

     ```shell
        npm i --save @web3auth/openlogin-adapter
        npm i --save @web3auth/phantom-adapter
        npm i --save @web3auth/torus-solana-adapter
     ```
