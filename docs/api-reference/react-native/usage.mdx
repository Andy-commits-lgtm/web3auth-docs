## Aviliable methods

---

### `login()`

Trigger the login flow that navigate the user to a browser model that allows them to login with the selected login provider.

### `logout()`

Trigger the logout flow on Android. This is a no-op on iOS since the iOS SDK does not have logout imeplemnted for technical reasons.

## Login

---

Trigger the login flow that navigate the user to a browser model that allows them to login with the selected login provider.

| Parameter     | Type                  | Mandatory                          | Description                                                                                                                                                                                           |
| ------------- | --------------------- | ---------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| provider      | enum of LoginProvider | Yes for bare React Native SDK only | Login provider, supported values are GOOGLE, FACEBOOK, REDDIT, DISCORD, TWITCH, APPLE, LINE, GITHUB, KAKAO, LINKEDIN, TWITTER, WEIBO, WECHAT, EMAIL_PASSWORDLESS, WEBAUTHN, JWT                       |
| loginProvider | enum of LoginProvider | Yes for Expo based SDK only        | Login provider, supported values are GOOGLE, FACEBOOK, REDDIT, DISCORD, TWITCH, APPLE, LINE, GITHUB, KAKAO, LINKEDIN, TWITTER, WEIBO, WECHAT, EMAIL_PASSWORDLESS, WEBAUTHN, JWT                       |
| redirectUrl   | String                | Yes on Expo based SDK only         | The redirectUrl that the Web3Auth login flow will use to redirect back to the app. It is not required to be specified in the bare React Native SDK since it will be specified in the `init()` method. |

### Sample Response

```json
{
  "ed25519PrivKey": "666523652352635....",
  "privKey": "0ajjsdsd....",
  "userInfo": {
    "aggregateVerifier": "tkey-google",
    "email": "john@gmail.com",
    "name": "John Dash",
    "profileImage": "https://lh3.googleusercontent.com/a/Ajjjsdsmdjmnm...",
    "typeOfLogin": "google",
    "verifier": "torus",
    "verifierId": "john@gmail.com"
  }
}
```

### Example

#### Bare React Native SDK

```typescript
const state = await OpenloginReactNativeSdk.login({
  provider: LoginProvider.GOOGLE,
});
```

#### Expo based SDK

```typescript
const scheme = "openloginexposdkexampleexpo";

const resolvedRedirectUrl =
  Constants.appOwnership == AppOwnership.Expo || Constants.appOwnership == AppOwnership.Guest
    ? new URL(Linking.createURL("openlogin", {}))
    : new URL(Linking.createURL("openlogin", { scheme: scheme }));

const state = await openlogin.login({
  loginProvider: LoginProvider.GOOGLE,
  redirectUrl: resolvedRedirectUrl,
});
```

## Logout

---

Trigger the logout flow on Android. This is a no-op on iOS since the iOS SDK does not have logout imeplemnted for technical reasons.

This method has no input and returns a `Promise` that resolves if the operation is successful, and rejects if the operation failed.

### Example

#### Bare React Native SDK

```typescript
await OpenloginReactNativeSdk.logout();
```

#### Expo based SDK

```typescript
await openlogin.logout();
```
